# Frontend репозитой проекта looQA

### Технологии

- **React 19.2.4**
- **TypeScript 5.9.3**
- **Vite 7.3.1**
- **React Router DOM 7.13.1**

### Библиотеки UI

- **Framer-Motion 12.35.1**
- **Ant-Design 6.3.1**

### Управление состоянием

- **MobX 6.15.0**

### Инструменты разработки

- **ESLint 10.0.3**
- **Prettier 3.8.1**

### Стилизация

- **CSS Modules**

## Структура проекта

```
src/
├── components/     # Переиспользуемые компоненты (глобальные)
├── pages/          # Страницы приложения
├── router/         # Конфигурация маршрутизации
├── store/          # Хранилища состояний
├── App.tsx         # Корневой компонент приложения
└── main.tsx        # Точка входа в приложение
```

## Скрипты

- `npm run dev` - запуск сервера для dev
- `npm run build` - сборка проекта для production
- `npm run lint` - проверка кода с помощью ESLint
- `npm run lint:fix` - исправление ошибок ESLint
- `npm run format` - форматирование кода с помощью Prettier

## Особенности конфигурации

### Prettier
- Использование одинарных кавычек
- Запятые в стиле ES5
- Ширина строки: 80 символов
- Отступы: 2 пробела
- Скобки на новой строке
- CamalCase в CSS modules

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Скачивание пакетов:
   ```bash
   npm install <lib_name> --registry https://registry.npmjs.org/
   ```
